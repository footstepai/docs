# Footstep Documentation

Source for the [Footstep API docs](https://docs.footstep.ai). Built with [Mintlify](https://mintlify.com).

## What's in here

- `api/` - REST API reference (routing, geocoding, AI, predict)
- `mcp/` - MCP server docs and tool reference
- `*.mdx` - Guides (quickstart, auth, errors, pricing, changelog)
- `api/openapi.json` - OpenAPI spec

## Local development

Install dependencies and start the preview server:

```sh
pnpm install
pnpm dev
```

This starts a local server at `http://localhost:3000` with hot reload.
